Modules:

Agent and Person
Agents are either persons or organizations. We use an Agent module, with Person as a subclass of Agent. If certain records apply to persons only, and not to agents in general, we declare this using appropriate axioms; see Axioms below.
Agents may have descriptions and external references as described in the respective module descriptions. Central for the Enslaved Ontology are the agent records associated with agents, this module is also described in detail further below.
